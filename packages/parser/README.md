# @menteka/parser

This library contains a lightweight feed (RSS/Atom) parser that is designt to run inside the browser. Designed for [Menteka](meneteka.com), but available for all that profit of it!

### Features:

- Zero dependencies
- Non-strict parsing
- Typescript support
- Open-source

### Supports:

- RSS 2
- Atom
- Enclosures
- Thumbnails
- Media detection
- Link detection
